# C++ Non-GUI Inventory System
### Author: Spencer Johns

### Description
This is a program written in C++20 that acts as the logic for an inventory system without the implementation of a graphical user interface to see the inventory.
The purpose of this project is to further my knowledge of the C++ language, classes, polymorphism, inheritance, file I/O, and OOP principles. I also just think
this will be a fun project to work on.
